# Logos de Technologies

Ce dossier contient les logos des technologies utilisées par Origin Studio.

## Logos requis (format SVG recommandé)

1. **infomaniak.svg** - Logo Infomaniak (hébergement suisse)
2. **react.svg** - Logo React
3. **nextjs.svg** - Logo Next.js
4. **tailwind.svg** - Logo Tailwind CSS
5. **nodejs.svg** - Logo Node.js
6. **postgresql.svg** - Logo PostgreSQL
7. **github.svg** - Logo GitHub

## Style des logos

Pour maintenir le design minimaliste et épuré du site :

- **Format** : SVG (vectoriel) recommandé
- **Couleur** : Versions monochromes (blanches ou grises) préférées
- **Taille** : Les logos seront automatiquement redimensionnés à une hauteur de 48px
- **Effet** : Grayscale par défaut, couleur au survol
